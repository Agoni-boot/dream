# 航空梦想漂流瓶

一个围绕“飞行梦想”设计的沉浸式单页互动网页。

用户可以把关于飞行的梦想、问题或心事写进漂流瓶，点击“发射”后，漂流瓶会进入流动的银河漩涡。页面会基于内容给出云端回信，访客也可以打开别人的漂流瓶继续留言互动。

## 项目简介

这个项目希望把“表达飞行梦想”从普通输入框和留言板，变成一个更有情绪、更有氛围、更有互动感的网页体验。

它不是传统的表单页，而是一幅会流动的画：

- 星河背景持续流动
- 漂流瓶沿漩涡轨迹卷入中心并循环出现
- 越靠近中心，漂流瓶越小、越亮、速度越快
- 用户自己的漂流瓶会收到与内容相关的云端回信
- 其他访客可以继续查看和留言，形成后续互动

## 核心亮点

- 沉浸式视觉：以银河、航迹、发光漂流瓶和卡通人物构成主场景
- 动态漩涡：漂流瓶沿弧线轨迹流动，具备景深、缩放、中心吸附和局部避让
- 内容有回应：根据用户输入内容生成相关云端回复，而不是固定文案
- 社区互动：支持查看他人漂流瓶、打开详情、继续留言
- 单文件实现：主页面使用原生 HTML + CSS + JavaScript，无需构建工具
- 本地持久化：使用 `localStorage` 保留瓶子与留言数据

## 使用场景

- 航空主题活动互动页
- 航空专业或航校展示页
- 青少年航空科普体验页
- 品牌活动或展览互动装置页
- 毕业季、开学季、节日主题心愿墙

## 技术实现

- 前端：原生 `HTML + CSS + JavaScript`
- 动效：`CSS Animation + requestAnimationFrame`
- 背景：`canvas` 渲染银河、离子流和旋涡氛围
- 数据：`localStorage`
- 架构：单页面、无后端、可直接本地运行

## 运行方式

### 方式一：直接打开

直接在浏览器中打开：

```text
aviation_dream_bottle.html
```

### 方式二：本地静态服务

如果浏览器环境限制本地文件访问，建议使用仓库里的 PowerShell 静态服务脚本：

```powershell
./serve-static.ps1
```

然后访问：

```text
http://127.0.0.1:8123/aviation_dream_bottle.html
```

## 主要功能

- 写下飞行梦想、问题或心愿
- 点击“发射”触发人物与漂流瓶动画
- 云端生成与内容相关的回信
- 打开银河中的漂流瓶查看详情
- 给别人的漂流瓶留言
- 查看自己的漂流瓶和后续互动

## 项目价值

- 解决普通留言页互动弱、氛围弱的问题
- 让航空主题内容从“信息展示”变成“情绪体验”
- 把梦想表达、内容回信和他人互动融合成一个完整场景
